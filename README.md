# Portif-lio-Pessoal-Simples
Página de apresentação pessoal moderna, com menu, animação de texto automática, botões e decoração de círculos coloridos. Desenvolvido com HTML, CSS e JavaScript, totalmente responsivo para celulares, tablets e computadores.

## ✨ Funcionalidades
- Menu de navegação e caixa de pesquisa
- Texto que escreve e apaga automaticamente
- Botões com cores diferentes
- Elementos decorativos circulares
- Layout adaptável
- Código organizado e fácil de modificar
